PS4 Summative 1
By Jasper Lyons & James Monk

Steps to run application:
	-Open PS4 Neighbourhood app, connect to a PS4
	-Open the remove viewer app & start audio/video streaming
	-In the release folder, drag the "PS4_ClearScreen_JL.elf" file onto the connected PS4
	-Click "Open" on the prompt, the application should now play on the remote viewer.

Control Scheme:
	D-Pad:
		Left & Right: Rotate sphere left/right
		Up & Down: Rotate cube up/down

	Analog sticks:
		Left:
			Move Forward/Back/Left/Right. (Relative to where you are looking)
			
		Right:
			Look Up/Down/Left/Right
